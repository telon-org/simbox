## О проекте

**SimBox** — это система управления GSM SIM-боксом для интеграции с Asterisk PBX. Система предназначена для маршрутизации голосовых вызовов через сеть GSM с использованием множественных USB-модемов и программируемых SIM-карт.

### Основное назначение

- **GSM-терминация** — маршрутизация вызовов через сеть GSM
- **Мультисим-адаптеры** — поддержка программируемых SIM-карт (Type 9 STK, Type 16 SuperSIM)
- **Мультиоператор** — поддержка 9 операторов связи (Россия, Украина, Беларусь)
- **Автоматизация** — голосовое распознавание, авто-блокировка, SMS-рассылки
- **Централизованное управление** — синхронизация с центральным сервером (simserver)

## Ключевые возможности

### 📱 Управление SIM-картами

- **KI-программирование** — загрузка криптографических ключей на мультисим-адаптеры
- **Баланс и тарифы** — отслеживание баланса, тарифов, номеров
- **USSD-запросы** — автоматические запросы баланса через USSD
- **Синхронизация** — отправка статистики на центральный сервер

### 📶 Управление модемами

- **Поддержка Huawei** — E1550, E173, E303, E353
- **AT-команды** — выполнение команд через Asterisk chan_dongle
- **Смена IMEI** — изменение IMEI модема
- **USB-хабы** — управление питанием через USB-хабы
- **Диагностический режим** — перепрошивка модемов

### 📞 Маршрутизация вызовов

- **Зонная маршрутизация** — правила маршрутизации по операторам/регионам
- **Спецрежимы** — VIP, тестовые, навигационные вызовы
- **Запись разговоров** — запись с конвертацией в FLAC
- **Входящие/исходящие** — полная обработка вызовов

### 🤖 Автоматизация и AI

- **Распознавание речи** — Yandex SpeechKit для транскрибации звонков
- **Автопилот** — автоматические действия (ввод PIN, активация)
- **SMS-автоматизация** — рассылки, запросы баланса
- **Авто-блокировка** — блокировка SIM по статистике (DATT, ACDL)
- **Watchdog** — мониторинг Asterisk с авто-перезагрузкой

### 🌐 Веб-интерфейс

- **Админ-панель** — управление SIM, модемами, вызовами
- **Биллинг** — статистика затрат по направлениям
- **Мониторинг** — состояние системы в реальном времени
- **Диагностика** — отладочная информация

## Установка

### Требования

- **ОС**: Linux (Debian/Ubuntu)
- **Asterisk**: 11 или 18 с модулем chan_dongle
- **PHP**: 5.x или 7.x с расширениями curl, xml
- **USB-хабы**: Terminus с поддержкой управления питанием
- **sox**: Для обработки аудио

### Конфигурация

1. **Настройка путей** — отредактируйте `config.sh`
2. **Asterisk** — настройте `extensions/extensions_dial.conf`
3. **Центральный сервер** — укажите `simserver:8122` в настройках
4. **Права доступа** — настройте `sudo` для выполнения команд

## Безопасность

### ⚠️ Критические замечания

1. **KI-ключи** передаются по HTTP без шифрования
2. **API-ключ Yandex** захардкожен в коде (`ai/recog/dorecog.php`)
3. **Нет аутентификации** для центрального сервера
4. **Веб-интерфейс** без видимой аутентификации
5. **Файлы KI** хранятся в открытом виде

### Рекомендации

- Включите HTTPS для связи с центральным сервером
- Реализуйте аутентификацию в веб-интерфейсе
- Шифруйте файлы с KI-ключами
- Настройте правильный `sudoers` с ограничениями
